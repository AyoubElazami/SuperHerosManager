# 🦸‍♂️ SuperHeroManager

Bienvenue dans le dépôt du projet **SuperHeroManager**. Il s'agit d'une application Full-Stack (MERN / Vite) développée dans le cadre de l'évaluation du module Technologie du Web.

🔗 **Lien du dépôt GitHub :** [https://github.com/AyoubElazami/SuperHerosManager.git](https://github.com/AyoubElazami/SuperHerosManager.git)

---

## 🛠️ Technologies Utilisées
- **Backend :** Node.js, Express.js, TypeScript, MongoDB, Mongoose, JWT, Multer
- **Frontend :** React 19, Vite, TypeScript, TailwindCSS v4, React Router v7, Formik, Yup

## 🚀 Comment lancer le projet en local

### 1. Cloner le dépôt
```bash
git clone https://github.com/AyoubElazami/SuperHerosManager.git
cd SuperHerosManager
```

### 2. Démarrer le Backend
Ouvrez un terminal et tapez :
```bash
cd backend
npm install
npm run dev
```
*(Le serveur démarrera sur http://localhost:5000)*

### 3. Démarrer le Frontend
Ouvrez un second terminal et tapez :
```bash
cd frontend
npm install
npm run dev
```
*(L'application React démarrera sur http://localhost:5173)*
